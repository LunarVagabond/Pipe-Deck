# Pipe Deck Documentation

Pipe Deck is the Linux Audio Control Center.

Product and technical documentation is maintained in this directory.

## Feature Acceptance Filter

Every proposed feature must answer **yes** to:

- Does this make Linux audio easier to understand and manage?

If the answer is no, the feature should be refined or dropped.
