# Pipe Deck Documentation

Pipe Deck is an open-source **Linux audio mixer and routing control center** for **PipeWire**.

This directory holds product and technical documentation. It is structured for migration to the [GitHub Wiki](https://github.com/pipedeck/pipe-deck/wiki); see [Wiki publishing](project/Wiki) for the checklist and sync steps.

**Wiki landing page:** [Home](Home.md) (use [Home](Home) on the wiki).

## Feature acceptance filter

Every proposed feature must answer **yes** to:

- Does this make Linux audio easier to understand and manage?

If the answer is no, the feature should be refined or dropped.
