## NOTICE

Flatpak support is being removed from Pipe Deck right now (July 20, 2026) due to the security restrictions on the sandbox flatpak's run in. This does **NOT** mean we will never bring it back but the overhead of maintaining it right now is out of scope for the goals of this project.

If you are a contributor who favors flatpak over App Images please feel free to get flatpak's working and then delete this README. We are leaving the manifest in tact for any brave enough to wire the flatpak!
